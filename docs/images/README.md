# Imágenes del Proyecto

Esta carpeta contiene las capturas de pantalla y diagramas del proyecto SIPI Modern.

## Estructura Recomendada

- `dashboard-admin.png` - Captura del dashboard del administrador
- `students-management.png` - Captura de la gestión de estudiantes
- `teachers-management.png` - Captura de la gestión de maestros
- `subjects-management.png` - Captura de la gestión de materias
- `groups-management.png` - Captura de la gestión de grupos
- `login-page.png` - Captura de la página de login (opcional)
- `teacher-dashboard.png` - Captura del dashboard del maestro (opcional)
- `student-dashboard.png` - Captura del dashboard del estudiante (opcional)

## Formato

- **Formato**: PNG o JPG
- **Resolución recomendada**: 1920x1080 o superior
- **Tamaño máximo**: 2MB por imagen
- **Nomenclatura**: Usar guiones bajos o guiones, en minúsculas

## Cómo Agregar Imágenes

1. Captura la pantalla del sistema
2. Guarda la imagen en esta carpeta con un nombre descriptivo
3. Actualiza el README.md principal con la referencia a la imagen
4. Asegúrate de que la imagen esté en el repositorio (no en .gitignore)
